# Job Tracker App

A full-stack job tracking application to manage job applications easily.

## Tech Stack
- Frontend: React
- Backend: Flask
- Database: SQLite
- Authentication: JWT

## Features
- User Registration
- User Login
- Add Job
- View Jobs
- Update Job Status
- Delete Job
- Logout

## How to Run the Project

### Backend
1. Open terminal
2. Go to backend folder
3. Run:
   python app.py

### Frontend
1. Open terminal
2. Go to frontend folder
3. Run:
   npm install  
   npm start
